# node-http-gammu

> A http API that exposes basic functionality of
> [wammu/gammu](https://wammu.eu/) for sending and
receiving SMS.

## Prerequisites 

### Creating a local copy of `.env`

The following fields must be configured:

```
SERVER_PORT=5000
```
