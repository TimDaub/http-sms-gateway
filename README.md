# http-sms-gateway

> A server for sending and receiving SMS via HTTP.

## Prerequisites 

### Creating a local copy of `.env`

The following fields must be configured:

```
SERVER_PORT=5000
DB_PATH=http-sms-gateway.db
SQLITE_SCHEMA_PATH=src/sql/schema.sql
BEARER_TOKEN=abc
```
